# Gestion de Stock - Cosumar

Application de gestion de stock et de commandes développée dans le cadre d'un projet de stage.

## Fonctionnalités

- **Gestion du catalogue** : Ajout, modification et suppression de produits et catégories.
- **Gestion des commandes** : Tunnel d'achat client et suivi des commandes côté administrateur.
- **Suivi des stocks** : Mise à jour en temps réel et alertes de rupture de stock.
- **Dashboard** : Visualisation des statistiques clés de l'activité.

## Technologies utilisées

- **Frontend** : React, TypeScript, Vite
- **Styling** : Tailwind CSS, shadcn/ui
- **Backend** : Supabase (Database & Auth)
- **Tests** : Vitest (Unitaires/Intégration), Playwright (E2E), k6 (Performance)

## Installation et Lancement

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/Salaheddine-Merchich/Gestion-de-stock.git
   ```

2. Installer les dépendances :
   ```bash
   npm install
   ```

3. Lancer l'application en mode développement :
   ```bash
   npm run dev
   ```

4. Lancer les tests :
   ```bash
   npm run test
   ```
